Supermini2332022-04-292025-09-19

Т»ЈТгАAndroidТќ░уЅѕТюгтЈЉтИЃтљј№╝їжѓБуДЇТ╝ФжЋ┐уГЅтЙЁтЇ┤У┐ЪУ┐ЪТћХСИЇтѕ░OTAТЏ┤Тќ░уџёуёдУЎЉ№╝їProject TrebleТГБТў»СИ║С║єУДБтє│У┐ЎСИђуЌЏуѓ╣УђїућЪсђѓ

тЅЇУ»Ю

тцџт╣┤ТЮЦ№╝їAndroidу│╗у╗ЪТЏ┤Тќ░у╝ЊТЁбжЌ«жбўСИђуЏ┤тЏ░ТЅ░уЮђућеТѕитњїтѕХжђатЋєсђѓТ»ЈтйЊGoogleтЈЉтИЃТќ░уЅѕТюг№╝їУ«ЙтцЄтѕХжђатЋєтњїУі»уЅЄСЙЏт║ћтЋєжЃйжюђУдЂУ┐ЏУАїТ╝ФжЋ┐УђїтцЇТЮѓуџёжђѓжЁЇУ┐ЄуеІсђѓ

У┐Ўт»╝УЄ┤AndroidУ«ЙтцЄтЇЄу║ДујЄжЋ┐ТюЪСйјУ┐и№╝їућџУЄ│2025т╣┤С╗ЇТюЅ17%уџёУ«ЙтцЄТЌаТ│ЋтЈіТЌХУјитЈќт«ЅтЁеУАЦСИЂсђѓ

2017т╣┤№╝їGoogleтюеAndroid 8.0СИГт╝ЋтЁЦС║єтљЇСИ║Project TrebleуџёТеАтЮЌтїќТъХТъёУ«ЙУ«А№╝їУ┐ЎТў»Androidу│╗у╗ЪТъХТъёУЄфУ»ъућЪС╗ЦТЮЦТюђжЄЇтцДуџётЈўжЮЕС╣ІСИђ№╝їТЌетюеС╗јТа╣ТюгСИіУДБтє│у│╗у╗ЪТЏ┤Тќ░у╝ЊТЁбуџёжЌ«жбўсђѓ

01 С╝ау╗ЪAndroidТЏ┤Тќ░уџётцЇТЮѓТхЂуеІ

тюеProject TrebleС╣ІтЅЇ№╝їAndroidу│╗у╗ЪТЏ┤Тќ░Тў»СИђСИфтцЇТЮѓСИћУђЌТЌХуџёУ┐ЄуеІсђѓждќтЁѕ№╝їGoogleт░єТќ░у│╗у╗ЪТ║љуаЂтЈЉтИЃУЄ│AOSP№╝ѕAndroidт╝ђТћЙТ║љС╗БуаЂжА╣уЏ«№╝Ѕсђѓ

ТјЦуЮђ№╝їУі»уЅЄтѕХжђатЋє№╝ѕтдѓжФўжђџсђЂСИЅТўЪсђЂУЂћтЈЉуДЉуГЅ№╝ЅжюђУдЂт»╣Т║љуаЂУ┐ЏУАїжњѕт»╣ТђДС┐«Тћ╣№╝їС╗ЦУ«ЕУЄфт«ХУі»уЅЄУЃйтцЪтюеТќ░уЅѕТюгAndroidу│╗у╗ЪСИіТГБтИИтљ»тіетњїУ┐љУАїсђѓ

уёХтљј№╝їУі»уЅЄтѕХжђатЋєт░єС┐«Тћ╣тљјуџёТ║љуаЂС║цу╗ЎOEMтјѓтЋє№╝їтјѓтЋєТа╣ТЇ«УЄфти▒уџёжюђТ▒ѓУ┐ЏСИђТГЦт«џтѕХТќ░у│╗у╗Ъ№╝їтіатЁЦуЅ╣УЅ▓тіЪУЃйтњїжбёУБЁт║ћућесђѓ

Тюђтљј№╝їу╗ЈУ┐ЄСИђу│╗тѕЌТхІУ»Ћтљј№╝їТќ░уЅѕу│╗у╗ЪТЅЇУЃйжђџУ┐ЄOTAТќ╣т╝ЈТјежђЂу╗ЎућеТѕисђѓУ┐ЎСИфУ┐ЄуеІТХЅтЈітцџТќ╣тЇЈСйю№╝їС╗╗СйЋСИђуј»тЄ║уј░т╗ХУ┐ЪжЃйС╝џт»╝УЄ┤ТЏ┤Тќ░ТјежђЂу╝ЊТЁбсђѓ

02 Project TrebleуџёТаИт┐ЃжЮЕТќ░

Project TrebleТюђтЁ│жћ«уџётЈўтїќТў»т╝ЋтЁЦС║єvendor interface№╝ѕСЙЏт║ћтЋєТјЦтЈБ№╝Ѕ уџёТдѓт┐х№╝їт░єVender Implementation№╝ѕућ▒Уі»уЅЄтѕХжђатЋєу╝ќтєЎуџёУ«ЙтцЄСИЊт▒ът║Ћт▒ѓУй»С╗Х№╝ЅС╗јAndroidу│╗у╗ЪТАєТъХСИГтЅЦуд╗тЄ║ТЮЦсђѓ

У┐ЎуДЇТъХТъёжЄЇу╗ётѕЏт╗║С║єСИђСИфуе│т«џуџёТјЦтЈБ№╝їСйЇС║јAndroidТЊЇСйюу│╗у╗ЪТАєТъХтњїУ«ЙтцЄуџёуЅ╣т«џуАгС╗Ху╗ёС╗ХС╣ІжЌ┤сђѓжђџУ┐ЄУ┐ЎСИфТјЦтЈБ№╝їAndroidТАєТъХтЈ»С╗ЦуІгуФІС║јУ«ЙтцЄуЅ╣т«џуџёт║Ћт▒ѓУй»С╗ХУ┐ЏУАїТЏ┤Тќ░сђѓ

СИ║С║єС┐ЮУ»ЂтЁ╝т«╣ТђД№╝їGoogleУ┐ўт╝ЋтЁЦС║єVendor Test Suite№╝ѕVTS№╝ЅТхІУ»ЋтЦЌС╗Х№╝їу▒╗С╝╝С║јтЁ╝т«╣ТђДТхІУ»ЋтЦЌС╗Х№╝ѕCTS№╝Ѕ№╝їућеС║јт»╣vendor interfaceуџётљѕуљєТђДтњїтЁ╝т«╣ТђДУ┐ЏУАїТБђжфїсђѓ

03 TrebleТъХТъёуџётЁ│жћ«ТіђТю»

Project Trebleт╝ЋтЁЦС║єтЄажА╣жЄЇУдЂТіђТю»тѕЏТќ░№╝џHIDL№╝ѕHALТјЦтЈБт«џС╣ЅУ»ГУеђ№╝ЅућеС║јТїЄт«џHALтњїтЁХС╗ќу╗ёС╗ХС╣ІжЌ┤уџёТјЦтЈБ№╝ЏУ«ЙтцЄТаЉтЈатіат▒ѓ№╝ѕDTO№╝Ѕт░єУ«ЙтцЄТаЉтѕєтЅ▓СИ║СИ╗DTтњїтЈатіаDT№╝їСй┐У«ЙтцЄТаЉТЏ┤т«╣ТўЊТЏ┤Тќ░тњїС┐«Тћ╣№╝ЏСЙЏт║ћтЋєтјЪућЪт╝ђтЈЉтЦЌС╗Х№╝ѕVNDK№╝ЅТЈљСЙЏС║єСИђу╗ёУ«ЕСЙЏт║ћтЋєт«ъуј░тЁХHALуџёСИЊућет║Њ№╝ЏСЙЏт║ћтЋєТјЦтЈБт»╣У▒А№╝ѕVINTF№╝ЅућеС║јТ▒ЄТђ╗У«ЙтцЄуЏИтЁ│С┐АТЂ»т╣ХжђџУ┐ЄтЈ»ТЪЦУ»буџёAPIТЈљСЙЏУ┐ЎС║ЏС┐АТЂ»сђѓ

HIDLтѕєСИ║СИцуДЇТеАт╝Ј№╝џPassthrough№╝ѕуЏ┤жђџт╝Ј№╝ЅтњїBinderized№╝ѕу╗Љт«џт╝Ј№╝ЅсђѓуЏ┤жђџт╝ЈHALСИ╗УдЂућеС║јтЁ╝т«╣ТЌДуЅѕHAL№╝їУђїу╗Љт«џт╝ЈHALтѕЎУ┐љУАїтюеуІгуФІуџёУ┐ЏуеІСИГ№╝їжђџУ┐ЄBinderТю║тѕХСИјТАєТъХжђџС┐Асђѓ

04 т»╣у│╗у╗ЪТЏ┤Тќ░уџёт«ъжЎЁтй▒тЊЇ

Project TrebleТъЂтцДтю░у«ђтїќС║єAndroidТЏ┤Тќ░ТхЂуеІсђѓтюеТќ░ТъХТъёСИІ№╝їGoogleтЈЉтИЃТќ░уЅѕТюгAOSPтљј№╝їOEMтјѓтЋєтЈ»С╗ЦуЏ┤ТјЦтЪ║С║јAТ║љуаЂУ┐ЏУАїт«џтѕХ№╝їТЌажюђуГЅтЙЁУі»уЅЄтѕХжђатЋєжђѓжЁЇсђѓ

Уі»уЅЄтѕХжђатЋєу╝ќтєЎуџёУ«ЙтцЄСИЊт▒ът║Ћт▒ѓжЃетѕєтѕЎућ▒GoogleтњїУі»уЅЄтјѓтЋєтЁ▒тљїу╗┤Тіц№╝їт╣ХУбФУ«ЙУ«АСИ║СИЇтєЇжЮбтљЉуЅ╣т«џуЅѕТюгуџёAndroidу│╗у╗Ъсђѓ

У┐ЎТёЈтЉ│уЮђOEMтјѓтЋєтЈ»С╗ЦТЏ┤т┐Фтю░жђѓжЁЇтњїтЈЉтИЃТќ░уЅѕТюгAndroidу│╗у╗Ъ№╝їТўЙУЉЌжЎЇСйјт╝ђтЈЉТѕљТюгтњїтцЇТЮѓт║д№╝їтљїТЌХт╗ХжЋ┐С║єУ«ЙтцЄућЪтЉйтЉеТюЪсђѓ

Google AndroidУй»С╗ХтиЦуеІжЃежЌетЅ»Тђ╗уљєDavid BurkeУАеуц║№╝їProject Trebleт░єРђюУ«ЕтѕХжђатЋєС╗ЦТЏ┤СйјуџёТѕљТюгТЏ┤Уй╗ТЮЙсђЂТЏ┤т┐ФжђЪтю░т░єУ«ЙтцЄТЏ┤Тќ░тѕ░Тќ░уЅѕAndroidу│╗у╗ЪРђЮсђѓ

05 т»╣уггСИЅТќ╣ROMт╝ђтЈЉуџётй▒тЊЇ

Project TrebleСИЇС╗Ёт»╣т«ўТќ╣ТЏ┤Тќ░ТюЅуЏі№╝їС╣ЪСИ║уггСИЅТќ╣ROMт╝ђтЈЉтИдТЮЦС║єжЮЕтЉйТђДтЈўтїќсђѓТћ»ТїЂTrebleуџёУ«ЙтцЄтдѓТъюТЃ│УдЂжбёУБЁТѕќтЇЄу║ДAndroid 8.0№╝їжЃйт┐ЁжА╗тЁѕу╗ЈУ┐ЄProject TrebleУ«цУ»Ђсђѓ

У┐ЎТёЈтЉ│уЮђУ«ЙтцЄтюеOEMтјѓтЋєт»╣AOSPТ║љуаЂУ┐ЏУАїт«џтѕХтЅЇ№╝їт░▒ти▓у╗ЈУЃйтцЪУ┐љУАїAOSPу│╗у╗Ът╣ХжђџУ┐ЄCDDсђЂCTSсђЂVTSуГЅСИђу│╗тѕЌТхІУ»Ћсђѓ

т╝ђтЈЉУђЁжђџУ┐ЄAOSPС╗БуаЂу╝ќУ»ЉуџёROMтЄаС╣јтЈ»С╗ЦтюеС╗╗СйЋТћ»ТїЂProject TrebleуџёТќ░ТЅІТю║СИіУ┐љУАї№╝їуггСИЅТќ╣ROMжђѓжЁЇтиЦСйютЈўтЙЌт╝ѓтИИу«ђтЇЋсђѓ

тЃЈLineage OSУ┐ЎТаиуџёУЄфт«џС╣ЅROMтЈ»С╗ЦТЏ┤т«╣ТўЊтю░жђѓжЁЇтљёуДЇУ«ЙтцЄ№╝їТъЂтцДтю░СИ░т»їС║єAndroidућЪТђЂу│╗у╗ЪуџётцџТаиТђДсђѓ

06 уј░т«ъТїЉТѕўСИјт▒ђжЎљТђД

т░йу«АProject TrebleтИдТЮЦС║єтиетцДТћ╣У┐Џ№╝їСйєAndroidубјуЅЄтїќжЌ«жбўт╣ХТюфт«їтЁеУДБтє│сђѓDavid BurkeТЏЙтЮдУеђ№╝їтЃЈСИГтЏйсђЂтЇ░т║дуГЅТќ░тЁ┤тИѓтю║ТюђтЦйуџёТЅІТю║т╣ХСИЇСИђт«џжЄЄућеТюђТќ░уЅѕу│╗у╗Ъ№╝їУђїТў»жђЅТІЕуЏИт»╣уе│т«џсђЂТѕљуєЪуџёуЅѕТюгсђѓ

У┐ЎуДЇтИѓтю║уГќуЋЦти«т╝ѓТёЈтЉ│уЮђСИ╗УдЂТхЂжђџуЅѕТюгС╗ЇтЈ»УЃйтГўтюети«т╝ѓсђѓ2025т╣┤уџёТЋ░ТЇ«ТўЙуц║№╝їС╗ЇТюЅ17%уџёAndroidУ«ЙтцЄТЌаТ│ЋтЈіТЌХУјитЈќт«ЅтЁеУАЦСИЂ№╝їубјуЅЄтїќжЌ«жбўСЙЮуёХтГўтюесђѓ

ТЪљС║Џтю░тї║тњїтИѓтю║уј»тбЃСИІ№╝їтјѓтЋєтЈ»УЃйС╗ЇуёХтЪ║С║јтЋєСИџУђЃжЄЈжђЅТІЕСИЇТЈљСЙЏТЏ┤Тќ░№╝їт░йу«АТіђТю»жџюубЇти▓у╗ЈтцДтцДжЎЇСйјсђѓ

Pixelу│╗тѕЌТЅІТю║СйюСИ║ждќТЅ╣Тћ»ТїЂProject TrebleуџёУ«ЙтцЄ№╝їујЄтЁѕСйЊжфїтѕ░С║єУ┐ЎСИђТіђТю»тИдТЮЦуџёС╝ўті┐сђѓ
